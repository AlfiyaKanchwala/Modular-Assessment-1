# Credit Card Fraud Detection using Data Mining Techniques

## Project Description

This project focuses on detecting fraudulent financial transactions using data mining techniques.

The project includes:

* Data preprocessing
* Exploratory Data Analysis
* Classification algorithms
* Clustering techniques
* Association rule mining

---

## Dataset Used

Dataset Name:

credit_fraud.csv.xlsx

The dataset contains transaction-related information used for fraud detection.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Mlxtend

---

## Algorithms Implemented

### Classification

* Decision Tree
* Naive Bayes

### Clustering

* K-Means Clustering

### Association Rule Mining

* Apriori Algorithm

---

## Performance Metrics Used

* Accuracy
* Precision
* Recall
* F1 Score
* Silhouette Score

1. Install required libraries
2. Open the Python file

## Project Outcome

The project successfully identified fraudulent transaction patterns using data mining techniques.

Decision Tree provided better balanced fraud detection performance compared to Naive Bayes.
